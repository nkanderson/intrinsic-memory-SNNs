# Intrinsic Memory in Spiking Neural Networks (SNNs)

This work focuses on investigation into the feature of intrinsic memory in neurons used to create SNNs. Intrinsic memory may be modeled in a number of different ways, with fractional-order differentiation representing biologically plausible dynamics. The goal is to identify tradeoffs when implementing intrinsic memory in a digital context, and determine what advantages, if any, may be obtained as compared to the more standard Leaky Integrate-and-Fire (LIF) neuron model.

Work in [Design Space](./1_design_space/) begins with design space analysis at the neuronal level, specifically addressing constraints and requirements for representation of a neuronal history term contributing to intrinsic memory. In [Training and Simulation](./2_training_and_simulation/), a workflow has been created for training SNNs on the cart-pole task using the snnTorch library. Successful models are then quantized and validated in software simulation before being executed in hardware simulation using SystemVerilog and cocotb. The [Benchmarking on FPGA](./3_benchmarking_on_FPGA/) section culminates in synthesis and execution of the hardware models on an FPGA.
